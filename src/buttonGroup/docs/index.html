1. 只置ng-model(必需)不设置其他任何参数:纯展示

  2. 设置ng-model(必需),默认类型(radio),则每个子元素<button>必须设置btn-radio属性

    ng-model:{{radioModel}}

  3. 设置ng-model(必需),类型设置为checkbox,则每个子元素<button>必须设置btn-checkbox属性,并且btn-checkbox对应值应该为ng-model的key

    ng-model:{{checkModel}}
  4. 设置ng-model(必需),类型设置为checkbox,则每个子元素<button>必须设置btn-checkbox属性,并且btn-checkbox对应值应该为ng-model的key,

    size设置为larger|default|small|x-small

  5. ng-model(必需),类型设置为checkbox,则每个子元素<button>必须设置btn-checkbox属性,并且btn-checkbox对应值应该为ng-model的key,

    show-class设置为danger | warning | default | success | info ｜ primary

  6. ng-model(必需),类型设置为checkbox,则每个子元素<button>必须设置btn-checkbox属性,并且btn-checkbox对应值应该为ng-model的key,

    disabled设置为true | false,默认为false

  7. ng-model(必需),类型设置为checkbox,则每个子元素<button>必须设置btn-checkbox属性,并且btn-checkbox对应值应该为ng-model的key,

    checkboxTrue和checkboxFalse设置选中(不选)状态对应ng-model的值。默认为true｜false

    原始值ng-model对应值:{{checkModel}} 设置checkboxTrue和checkboxFalse后ng-model对应值:{{checkModelVal}}
